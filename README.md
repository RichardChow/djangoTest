# Django Backend Configuration Management

这是一个基于 Django 和 Django REST framework 的配置管理后端系统。

## 项目概述

该项目提供了一个 RESTful API 服务，用于管理和存储配置信息，包括升级类型、工作类型和环境-IP映射关系。

### 主要功能

- 配置的增删改查
- 支持多种升级类型管理
- 支持多种工作类型管理
- 环境和IP列表的映射关系管理
- 提供 API 和 Web 界面两种访问方式

## 技术栈

- Python 3.8
- Django 4.2.16
- Django REST framework
- SQLite3 数据库
